# Agent Prompts Library

This directory contains role-specific prompts to guide the AI for different types of tasks.

## Directory Structure
- `coding/`: Prompts for software engineering, specific languages, and frameworks.
- `architecture/`: Prompts for system design, database modeling, and high-level planning.
- `product/`: Prompts for product management, feature specification, and user experience.
- `quality/`: Prompts for testing, QA, and code review.
