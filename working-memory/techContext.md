# Tech Context

## Stack
-   **Frontend**: React 18+
-   **Build Tool**: Vite
-   **Language**: TypeScript
-   **Styling**: TailwindCSS
-   **Charts**: Recharts

## External Services
-   **Concept2 Logbook API**: OAuth2, REST.
-   **Google Sheets API**: OAuth2, REST.

## Configuration
-   Requires `.env` file with:
    -   `VITE_CONCEPT2_CLIENT_ID`
    -   `VITE_CONCEPT2_CLIENT_SECRET`
    -   `VITE_GOOGLE_CLIENT_ID`
